WHAT IS IT

Log5F is logging tool for Flash/Flex/AIR projects, it started in 2008 as a project that ports Log4J. Log5F support hierarchical loggers by package similar to Log4J. It does not provide any GUI to view logs, but can send logging data to many of them. Log5F independent from Flex SDK, it can be used in pure Flash projects starting from Flash Player 9.0
