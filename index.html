GITHUB: source code
JIRASTUDIO: wiki, issues, builds
MAVEN: artifacts
WHAT IS IT

We accustomed to debug our applications, during this paramount development phase, we find a most problems that exist in application, but not all. Sometimes application contains bugs after project release, and when it happened, the finding of these bugs can be a laborious process. To make this process more easy we use logging.
